
# Auto Schedule Extension

A lightweight chrome extension that scans and parses emails locally. Suggesting to add to the schedule with out any hassles. 




## Installation

Download all the files in one empty folder of your choice

Ex.
```
ASE v.x.x.x
 ├──background.js
 ├──content.js
 ├──popup.js
 ├── popup.html
 └── manifest.json
```

Head over to [chrome://extensions](chrome://extensions) and click "Load unpacked" and select the empty folder

## Features

- Light/dark mode toggle
- Selections of preferred calender 
- Highlight text and right click to parse specific info
